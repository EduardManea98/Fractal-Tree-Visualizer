About: Fractal Tree Visualizer

HOW TO OPEN:
Option 1) Open: https://editor.p5js.org/EduardManea98/sketches/4uHasfz9g
Option 2) Download the project and open it in browser.

There are 2 sliders:
First slider: Change branche's inclination
Second slider: Change size of the tree. 
	        i) Number of branches is increased if you make it bigger and opposite once you make it smaller.
	       ii) When you change the size, branches changes color also.
